# 🚀 Interactive Product Hotspots

An elegant and lightweight interactive UI component that adds animated hotspots to any product image. Designed for modern landing pages, product demos, and UI/UX showcases.

---

## ✨ Preview

Click or hover over animated dots to reveal feature descriptions.

> Perfect for showcasing product features in a clean, engaging way.

---

## 🧩 Features

- 🎯 Interactive hotspots with smooth animations  
- 💡 Tooltip-based feature explanations  
- 📱 Fully responsive and mobile-friendly  
- ⚡ Lightweight (no frameworks required)  
- 🎨 Clean, modern UI design  
- 🖱️ Supports hover (desktop) + click (mobile)

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Animations + Layout)  
- Vanilla JavaScript (No libraries)
